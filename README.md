# Shell Script Website Deployment Project

This project demonstrates how to deploy a static website on an AWS EC2 instance using shell scripting.

## Features
- Automated package update
- Apache web server installation
- Website files deployment using shell script
- Simple and beginner-friendly DevOps project

## Technologies Used
- Linux (Ubuntu)
- Shell Scripting (Bash)
- Git & GitHub
- AWS EC2
- Apache Web Server

## How to Run
1. Clone the repository
2. Give execute permission to the script
3. Run the deployment script

```bash
chmod 700 deploy.sh
./deploy.sh
